# Advantages of Git

1. **Speed** – Fast operations (commit, merge, branch).
2. **Distributed** – Each developer has a full copy of repo.
3. **Branching/Merging** – Easy to create and merge branches.
4. **Data Integrity** – Uses SHA-1 hash to secure history.
5. **Collaboration** – Works well with GitHub, GitLab, Bitbucket.
6. **Open Source** – Free to use, widely supported.

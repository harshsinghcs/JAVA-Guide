# Introduction to Git

- Git is a **distributed version control system**.
- Helps track changes in source code during software development.
- Allows multiple developers to work on the same project simultaneously.
- Created by **Linus Torvalds** in 2005 for Linux kernel development.

## Key Features
- Distributed system (every developer has a full copy of repo).
- Branching & merging support.
- Data integrity (SHA-1 hashing).
- Lightweight and fast.
